# 妈妈手机助手隐私政策

**最后更新日期：2026年2月10日**

妈妈手机助手（Family Care Assistant，以下简称“我们”）致力于保护您的隐私。本隐私政策旨在说明我们的移动应用程序（以下简称“本应用”）如何收集、使用和披露信息，以及您对这些信息拥有的选择权。

## 1. 我们收集和使用的信息

本应用旨在帮助子女远程协助年迈父母操作智能手机。为提供此功能，本应用需要某些敏感权限。

### 1.1 无障碍服务 API (Accessibility Service API)
**用途：** 本应用使用无障碍服务 API 允许受信任的远程用户（即“子女”或“协助者”）在此设备（即“父母”或“受助者”设备）上执行手势操作（如点击、滑动、返回/主页操作）。
**数据收集：**
*   **我们绝不会使用无障碍服务收集任何个人或敏感用户数据。**
*   该 API 仅严格用于接收来自已连接子女设备的控制指令并在本地执行。
*   我们不会读取、存储或传输您使用此服务输入的任何文本（如密码或信用卡号）。

### 1.2 媒体投影（屏幕录制）
**用途：** 本应用使用媒体投影 API (Media Projection API) 捕获此设备的屏幕内容，并将其实时流式传输到已连接的子女设备。
**数据收集：**
*   **仅限瞬时传输：** 屏幕内容通过我们的中继服务器实时传输到已连接的子女设备。
*   **不予存储：** 我们不会在服务器或设备上录制、保存或存储任何屏幕数据。数据是瞬时的，仅存在于活跃的远程支持会话期间。
*   **安全限制：** 根据 Android 系统安全规定，受 `FLAG_SECURE` 保护的敏感屏幕（如银行应用或密码输入框）在远程用户端将显示为黑屏或模糊画面。我们不会也无法绕过此限制。

### 1.3 设备信息
我们可能会收集最基本的设备信息（如设备型号和操作系统版本），仅用于建立稳定连接和优化视频流质量。

## 2. 数据传输方式

本应用使用中继服务器在父母设备和子女设备之间建立连接。
*   **仅作中继：** 服务器仅作为桥梁，在两台设备之间转发数据包（视频流和控制指令）。
*   **无持久化：** 中继服务器不存储任何视频流或控制日志。

## 3. 数据共享与披露

我们不会出售、交易或以其他方式向外界转移您的个人身份信息。
*   **点对点性质：** 屏幕数据**仅**共享给输入了您屏幕上显示的正确 6 位连接代码的特定子女设备。您完全掌控与谁共享屏幕。
*   **法律要求：** 如果法律要求，或者我们善意地认为必须采取此类行动以遵守法律义务时，我们可能会披露信息。

## 4. 安全性

我们实施适当的技术和组织措施来保护通过本应用传输的数据。然而，请注意，互联网上的传输方法并非 100% 安全。我们努力使用商业上可接受的手段保护您的信息，但不能保证其绝对安全。

## 5. 儿童隐私

虽然本应用是家庭护理工具，但并非针对 13 岁以下儿童。我们不会故意收集 13 岁以下儿童的个人信息。

## 6. 本隐私政策的变更

我们可能会不时更新我们的隐私政策。我们将通过在此页面发布新的隐私政策来通知您任何变更。建议您定期查看本隐私政策以了解任何变更。
